# The Sovereign Protocol

Voice-first interface for all 11 TrueSite family members.

## How it works

1. **Clap twice** or tap ADAMA to activate
2. **Speak** — Whisper transcribes your voice
3. **Active agent responds** in their ElevenLabs voice
4. **Switch agents** by saying "Hey Azari" / "Bring up Yahriel" / "Yahziel, real quick"
5. **Say "start the Sovereign Protocol"** to trigger the full family intro roll call
6. **Say "hey family, check in"** for a quick status from all agents

## Setup

```bash
cp .env.example .env
# Fill in OPENAI_API_KEY (for Whisper)
# ELEVENLABS_API_KEY is pre-filled

npm install
npm run dev
# Open http://localhost:5173
```

## Production build

```bash
npm run build
npm start
# Open http://localhost:4000
```
